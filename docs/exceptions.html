Exceptions¶

This module contains the exceptions used for the EveryAction client

exception everyaction.exception.EAException¶

Class of exceptions raised for EveryAction-related errors which are raised prior to sending HTTP requests. HTTP errors are raised as instances of the subclass EAHTTPException.

exception everyaction.exception.EAFindFailedException¶

Class of exceptions raised when a function cannot proceed because it expected to find data but failed.

exception everyaction.exception.EAHTTPException(response: requests.models.Response)¶

Class of exceptions raised if an error response was received for an EveryAction request.

Variables

  • response (Response) – The response with status code >= 400.

  • errors (Sequence[Error]) – List of EveryAction Error objects given in the response.

  • http_error (HTTPError) – The HTTPError associated with the error response.
